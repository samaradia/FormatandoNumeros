# Formatação de Valor em Java

Projeto simples em Java para praticar formatação de números.

## O que foi utilizado

- `Scanner`
- `String.format()`

## Objetivo

Aprender como formatar valores numéricos para exibição com casas decimais em Java.

## Como funciona

O programa:
- Recebe um valor decimal do usuário
- Formata o número com 2 casas decimais
- Exibe o valor formatado com "R$"

## Exemplo

Entrada:
```
10.5
```

Saída:
```
Valor formatado: R$ 10.50
```

## Tecnologias

- Java
